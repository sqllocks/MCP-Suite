# MCP-SUITE v3.0.0

**Production-ready MCP (Model Context Protocol) server suite with 28 specialized AI servers.**

## Quick Start

### Windows
```powershell
.\install-windows.ps1
.\start-mcp.bat
```

### macOS
```bash
./install-mac.sh
./start-mcp.sh
```

## What's Included

- **28 MCP Servers** with 36 specialized AI tools
- **SOC 2 Security** implementation
- **4 Profiles**: Personal, PathGroup, Beacon, EyeSouth
- **Auto-installation** for Windows & Mac
- **VS Code workspaces** pre-configured
- **PM2 process management** built-in
